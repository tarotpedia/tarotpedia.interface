# tarotpedia - interface for tarot card reading

tarotpedia is an open-source web application that provides an interactive interface for tarot card reading. It allows users to explore different tarot spreads, interpret card meanings, and gain insights through tarot readings.

## Features

- Interactive tarot card spreads
- Detailed card meanings and interpretations
- User-friendly interface

## Development

To set up the development environment for tarotpedia, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/tarotpedia/tarotpedia.interface
   cd tarotpedia.interface
   ```

2. Install dependencies:

   ```bash
   bun i
   ```

   > Make sure you have [Bun](https://bun.sh/) installed on your system.

3. Start the development server:

   ```bash
   bun dev
   ```

4. Open your browser and navigate to `http://localhost:3000` to view the application.

!!! note
You should create a `.env.local` file in the root directory to store environment variables. Refer to `.env.example` for the required variables.
